# mpvue-weui

`mpvue-weui` 是基于 `mpvue` 的一个小程序 UI 框架， 保留了 [weui](https://github.com/Tencent/weui) 的视觉规范，旨在提高小程序开发体验，同时让用户的使用感知更加统一。

# 相关说明

之前写了 [用 vue 写小程序，基于 mpvue 框架重写 weui](https://github.com/KuangPF/mpvue-weui),这个是在 mpvue 开源之初写的，开始的目的是想重写下 weui，然后写了一下踩坑文档。可能有点遗憾的是没有对其进行完整的封装，以至于在使用过程中就很不方便，在 issues 中也看到一些关于能不能封装成一个独立的 UI 框架的想法，加上大家也给了这个仓库这么多 :star: :star: ，因此决定将其封装城一个独立的 UI 框架：mpvue-weui。
